# To clone this project :
----------------------------------
### git clone https://github.com/alzaabi98/bestBlog.git
### cd bestblog
### composer install
### npm install
## create copy for .env  : cp .env.example .en
### php artisan key:generate
### create new database in your mysql server and then add it to .env file
### php artisan migrate
